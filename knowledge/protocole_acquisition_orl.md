# 1. Informations Générales

## 1.1 Objectif du protocole

Ce protocole standardise la réalisation des examens TEP CT FDG avec injection de produit de contraste iodé (PCI) dans le cadre du bilan d'extension des cancers ORL de stade avancé (III-IV).

## 1.2 Indications

| Localisation | Indication TEP CT | Référence |
|--------------|-------------------|-----------|
| **Cavité buccale** | Stades III-IV (staging, métastases) | EHNS-ESMO-ESTRO 2020 |
| **Oropharynx** | Stades III-IV (HPV+ : ganglions kystiques) | PMID 33239190 |
| **Hypopharynx** | Stades III-IV | EHNS-ESMO-ESTRO 2020 |
| **Larynx** | Stades III-IV | EHNS-ESMO-ESTRO 2020 |
| **Nasopharynx** | Tous stades (staging, réponse) | ESMO 2021 |
| **Sinus/Fosses nasales** | Stades avancés | ESMO 2025 |
| **Glandes salivaires** | Tumeurs de haut grade | ESMO 2022 |

## 1.3 Justification du CT injecté en ORL

### Recommandation clé (PMID 20652890)

> "In patients scheduled for ¹⁸F-FDG-PET/CT, **we strongly suggest performing a contrast-enhanced ¹⁸F-FDG-PET/CT**, which is not routine in most centers."

### Bénéfices spécifiques ORL

| Bénéfice | Application ORL |
|----------|-----------------|
| **Ganglions nécrotiques/kystiques** | Fréquents en HPV+ oropharyngé, peu FDG-avides |
| **Extension extranodale (ENE)** | CT injecté plus précis que PET seul (92% vs 74% spécificité) |
| **Envahissement vasculaire** | Différencier adénopathies des vaisseaux cervicaux |
| **Base du crâne** | Meilleure délimitation (si IRM non disponible) |
| **Examen unique** | Staging métabolique + morphologique |

### Niveau de preuve

| Source | Recommandation contrast-enhanced PET/CT |
|--------|----------------------------------------|
| EHNS-ESMO-ESTRO 2020 | Non mentionné spécifiquement |
| NCCN 2025 | Non mentionné spécifiquement |
| EANM v3.0 | Optionnel, selon protocoles locaux |
| **Études comparatives** | Recommandé (PMID 20652890, 26122372) |

# 2. Section Manipulateur

## 2.1 Vérifications Pré-Examen

### 2.1.1 Éligibilité Contraste Iodé

#### Questionnaire à la prise de RDV (ESUR 2025)

| Question | Si OUI |
|----------|--------|
| Maladie rénale connue (eGFR <60) ? | Demander créatinine/eGFR |
| Chirurgie rénale / rein unique / greffe ? | Demander créatinine/eGFR |
| HTA traitée ? | Demander créatinine/eGFR |
| Diabète ? | Demander créatinine/eGFR |

**Délai du bilan rénal (si indiqué):**
- Patient ambulatoire stable : ≤3 mois
- Patient hospitalisé / maladie aiguë : ≤7 jours

#### Contre-indications absolues au PCI

| Contre-indication | Action |
|-------------------|--------|
| Allergie sévère connue au PCI | CT non injecté OU prémédication après avis allergologue |
| Hyperthyroïdie manifeste non contrôlée | Reporter, avis endocrinologique |

#### Précautions particulières

| Situation | Conduite à tenir | Source |
|-----------|------------------|--------|
| **eGFR 30-44** | Hydratation préventive | ESUR 2025 |
| **eGFR <30** | Discussion bénéfice/risque | ESUR 2025 |
| **Metformine + eGFR >30** | **Continuer normalement** | EANM v3.0 |
| **Metformine + eGFR <30** | STOP à l'injection, contrôle 48h | EANM v3.0 |
| **Inhibiteurs SGLT2** | Omission 48h si préoccupation | EANM v3.0 |

### 2.1.2 Vérifications Jour J

#### Checklist arrivée patient

- [ ] **Jeûne ≥6 heures** (eau plate autorisée)
- [ ] **Glycémie** vérifiée : <11 mmol/L (200 mg/dL)
- [ ] **eGFR vérifié** si facteur de risque
- [ ] **Allergie au PCI** : questionnée et documentée
- [ ] **Metformine/SGLT2** : vérifier eGFR si concerné
- [ ] **Prothèses dentaires amovibles** : retirées si possible (artefacts)

## 2.3 Protocole d'Injection

### 2.3.1 Chronologie détaillée

```
PRÉPARATION
───────────────────────────────────────────────────────
T-30 min   │ Accueil patient, vérifications checklist
           │ Mesure glycémie capillaire
           │ Retrait prothèses dentaires amovibles
           │
T-20 min   │ Pose VVP (20G minimum)
           │ Test au sérum phy (10-20 mL)
           │
───────────────────────────────────────────────────────
INJECTION FDG
───────────────────────────────────────────────────────
T0         │ INJECTION FDG IV
           │ Rinçage 10-20 mL NaCl 0.9%
           │ Noter heure exacte d'injection
           │
T0→T60     │ Période d'uptake : repos au calme
           │ - Position allongée ou assise
           │ - ÉVITER DE PARLER (activité muscles cervicaux)
           │ - Température ambiante confortable
           │
───────────────────────────────────────────────────────
ACQUISITION
───────────────────────────────────────────────────────
T60 min    │ Installation sur table TEP CT
(±5 min)   │ Positionnement bras le long du corps
           │ (ORL : éviter artefacts bras relevés)
           │
           │ 1. TOPOGRAMME
           │
           │ 2. CT DIAGNOSTIQUE INJECTÉ
           │    - Injection PCI (voir 2.3.2)
           │    - Délai 50-70 sec (phase porto-veineuse)
           │    - Acquisition CT
           │
           │ 3. ACQUISITION TEP
           │    - Du vertex au mi-cuisse
           │
T+90 min   │ FIN EXAMEN
           │ Surveillance 15-30 min post-PCI
```

### 2.3.2 Injection Produit de Contraste Iodé

| Paramètre | Valeur | Source |
|-----------|--------|--------|
| **Type** | Iso- ou hypo-osmolaire non ionique | ESUR 2025 |
| **Débit** | **2.5 mL/s** | EANM v3.0 |
| **Volume** | Selon protocole local (80-120 mL) | - |
| **Température** | **37°C** | ESUR 2025 |
| **Phase** | **Porto-veineuse** | EANM v3.0 |
| **Délai** | **50-70 secondes** post-injection | EANM v3.0 |

### 2.3.3 Contraste Oral (optionnel)

> "Water or water-based contrast agents [...] provide improved image quality with reduced artefacts." (EANM v3.0)

**En ORL :** Généralement non nécessaire sauf si extension médiastinale/œsophagienne suspectée.

## 2.4 Positionnement Patient

| Élément | Recommandation ORL |
|---------|-------------------|
| **Position** | Décubitus dorsal |
| **Bras** | **Le long du corps** (éviter artefacts sur cou/thorax supérieur) |
| **Tête** | Neutre, légère extension si toléré |
| **Immobilisation** | Cale tête recommandée |

**Point d'attention :** Éviter mouvements de déglutition pendant acquisition cervicale.

---

## 2.5 Gestion des Incidents

### 2.5.1 Extravasation PCI (ESUR 2025)

| Sévérité | Action |
|----------|--------|
| **Légère** (<50 mL) | Élévation membre + glace |
| **Modérée** (50-150 mL) | Évaluation neurovasculaire |
| **Sévère** (>150 mL) | **AVIS CHIRURGICAL URGENT** |

### 2.5.2 Réaction Allergique (ESUR 2025)

**Réaction modérée-sévère :**
```
1. APPELER ÉQUIPE URGENCE
2. Oxygène 10-15 L/min
3. Adrénaline 0.5 mg IM
4. Remplissage NaCl 500 mL
5. Antihistaminique IV
```

# 4. Section Médecin Interprétant

## 4.1 Spécificités ORL

### 4.1.1 Ganglions nécrotiques/kystiques (HPV+)

**Contexte :** Les carcinomes oropharyngés HPV+ présentent fréquemment des métastases ganglionnaires kystiques, peu ou pas FDG-avides.

| Imagerie | Performance pour ganglions kystiques |
|----------|-------------------------------------|
| TEP seul | **Insuffisant** (faux négatifs) |
| CT injecté | Bonne détection |
| **TEP CT injecté** | **Optimal** |

> "Contrast-enhanced CT and contrast-enhanced ¹⁸F-FDG-PET/CT perform equally and better than non-enhanced ¹⁸F-FDG-PET/CT in detecting cystic lymph node metastases." (PMID 20652890)

**Conduite pratique :** En cas de carcinome oropharyngé (surtout HPV+), le CT injecté est particulièrement recommandé.

### 4.1.2 Extension Extranodale (ENE)

**Importance :** L'ENE est un facteur pronostique majeur (upstaging pN → N3b).

| Imagerie | Spécificité ENE | Précision ENE |
|----------|-----------------|---------------|
| **CT injecté** | **92.1%** | **88.6%** |
| TEP CT | 74.4% | 75.3% |

> Source : PMID 26122372

**Signes CT d'ENE :**
- Irrégularité des contours ganglionnaires
- Infiltration de la graisse péri-ganglionnaire
- Prise de contraste irrégulière

### 4.1.3 Classification ganglionnaire cervicale

| Niveau | Localisation | Tumeurs drainées |
|--------|--------------|------------------|
| I | Sous-mental, sous-mandibulaire | Cavité buccale, lèvre |
| II | Jugulaire supérieur | Oropharynx, nasopharynx, parotide |
| III | Jugulaire moyen | Oropharynx, hypopharynx, larynx |
| IV | Jugulaire inférieur | Hypopharynx, larynx, thyroïde |
| V | Triangle postérieur | Nasopharynx, cuir chevelu |
| VI | Compartiment central | Larynx, thyroïde |
| VII | Médiastin supérieur | Thyroïde, œsophage cervical |

---

## 4.2 Pièges et Faux Positifs

### 4.2.1 Uptake Physiologique ORL

| Structure | Caractéristiques | Piège |
|-----------|------------------|-------|
| **Amygdales palatines** | Uptake symétrique | Asymétrie = suspect |
| **Amygdale linguale** | Base de langue | Différencier de tumeur |
| **Anneau de Waldeyer** | Uptake physiologique variable | Corrélation clinique |
| **Muscles laryngés** | Si phonation pendant uptake | Éviter de parler |
| **Muscles masticateurs** | Si mastication | Éviter chewing-gum |
| **Graisse brune** | Cervicale, symétrique | Aspect typique, β-bloquants |

### 4.2.2 Pathologies Bénignes

| Pathologie | Piège |
|------------|-------|
| **Inflammation post-radique** | Hypermétabolisme persistant |
| **Infection dentaire/sinusienne** | Uptake intense focal |
| **Granulome/Sarcoïdose** | Adénopathies hypermétaboliques |
| **Thyroïdite** | Uptake thyroïdien diffus |

---

## 4.3 Pièges et Faux Négatifs

| Situation | Raison | Solution |
|-----------|--------|----------|
| **Ganglions kystiques HPV+** | Faible cellularité | CT injecté |
| **Petites lésions (<1 cm)** | Résolution spatiale | IRM |
| **Tumeurs bien différenciées** | Faible métabolisme | Corrélation histologie |
| **Post-traitement récent** | Nécrose, inflammation | Délai ≥12 semaines |

---

## 4.4 Interprétation par Localisation

### 4.4.1 Oropharynx (HPV+ et HPV-)

| Élément | Points d'attention |
|---------|-------------------|
| **Tumeur primitive** | Base de langue, amygdales, voile |
| **Ganglions** | Niveaux II-III-IV, **ganglions kystiques HPV+** |
| **ENE** | Facteur pronostique majeur |
| **Second primitif** | Poumon, œsophage (tabac-alcool) |

### 4.4.2 Cavité Buccale

| Élément | Points d'attention |
|---------|-------------------|
| **Tumeur primitive** | Langue mobile, plancher, gencive |
| **Envahissement osseux** | Mandibule (fenêtre os) |
| **Ganglions** | Niveaux I-II-III |

### 4.4.3 Larynx / Hypopharynx

| Élément | Points d'attention |
|---------|-------------------|
| **Tumeur primitive** | Extension trans-glottique, cartilages |
| **Cartilages** | Lyse vs sclérose (CT) |
| **Ganglions** | Niveaux II-III-IV-VI |

### 4.4.4 Nasopharynx

| Élément | Points d'attention |
|---------|-------------------|
| **Tumeur primitive** | Cavum, extension base du crâne |
| **Base du crâne** | Foramen ovale, sinus caverneux |
| **Ganglions** | Rétropharyngés, cervicaux bilatéraux |

### 4.4.5 Sinus / Fosses Nasales

| Élément | Points d'attention |
|---------|-------------------|
| **Tumeur** | Extension orbitaire, base du crâne |
| **Différentiel** | Rétention sinusienne vs tumeur |

### 4.4.6 Glandes Salivaires

| Élément | Points d'attention |
|---------|-------------------|
| **Parotide** | Nerf facial (IRM préférable) |
| **Sous-mandibulaire** | Extension plancher buccal |
| **Ganglions** | Intra-parotidiens, cervicaux |

---

## 4.5 Artefacts liés au Contraste

### 4.5.1 Artefacts de Correction d'Atténuation

| Situation | Effet | Solution |
|-----------|-------|----------|
| Bolus veineux jugulaire | Surestimation SUV | Vérifier NAC |
| Prothèses dentaires métalliques | Artefacts importants | Retrait si possible, NAC |

### 4.5.2 Conduite à tenir

> "Lesions found on AC-PET images should be checked on NAC-PET images if located adjacent to highly attenuating material." (EANM v3.0)
